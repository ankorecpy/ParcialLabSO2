@mainpage

Este proyecto implementa una comunicacion cliente-servidor, ya sea de forma local/remota. El objetivo de este proyecto es el encio de archivos regulares por parte del cliente al servidor, asi mismo, la recepción y almacenamiento de archivos de forma concurrente por parte del servidor.
El cliente envia el nombre del archivo, identificacion de estudiante(s), tamaño del archivo y su contenido.
El servidor recibe lo enviado por el cliente y lo almacena de forma que no se omita el almacenamiento de ningun archivo.
